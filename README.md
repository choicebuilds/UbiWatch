# JWatch

## Tech Stack

### This watch is entirely built from scratch. A significantly improved version of [BumWatch][BW]!

MCU: NORA-B106 (nRF5340) - Dual-core ARM Cortex-M33
Sensors: MAX30102 + MAX32664 (HR/SpO2), BMI270 (IMU)
Power: MAX17055 Fuel Gauge, 1.8V/3.3V dual-rail LDOs
Display: 1.28" Round SPI TFT (GC9A01)
Design Tools: Altium Designer (PCB), VS Code/nRF Connect SDK (Firmware), Solidworks (Chassis)
